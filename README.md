## 🎵 О проекте

**NullMusic** — это приложение для прослушивания музыки, созданное с упором на минимализм, полное отсутствие рекламы и максимальный комфорт. Слушайте то, что любите, без прерываний и ограничений.

## 🚀 Основные возможности

- **Никакой рекламы:** Чистый стриминг музыки без подписок и раздражающих вставок.
- **Офлайн-режим:** Скачивайте любимые треки, альбомы и плейлисты для прослушивания без интернета.
- **Импорт плейлистов:** Удобный перенос вашей медиатеки из Яндекс Музыки в пару кликов.
- **Синхронизированные тексты:** Точное отображение текстов песен слово в слово.
- **Распознавание музыки:** Встроенный аналог Shazam — узнавайте, какой трек сейчас играет рядом с вами.
- **Фоновое воспроизведение:** Музыка продолжает играть даже при выключенном экране.
- **Кастомизация:** Тонкая настройка интерфейса, поддержка динамических цветов (Material You) и темной темы.

## 📥 Установка

1. Перейдите в раздел **Releases** на GitHub.
2. Скачайте последнюю версию APK-файла (`Universal`).
3. Установите на свое Android-устройство (может потребоваться разрешение на установку из неизвестных источников).
